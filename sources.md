# 来源、更新与联盟链接政策

## 这个网站是什么

本站是独立的中文 BandwagonHost（搬瓦工）使用指南，不属于 IT7 Networks Inc. 或 BandwagonHost 官方网站。

## 事实来源优先级

1. BandwagonHost 实时订单与产品页面；
2. BandwagonHost 官方服务条款、知识库、SLA 和新闻；
3. KiwiVM 或 Client Area 实际显示的账户级权益；
4. 可复现的网络、页面或接口测试。

价格、库存、产品 ID、机房和套餐权益属于易变信息。文章注明的“最近核对”只代表该日期的检查结果，购买前仍应以官方结算页为准。

## 如何处理不确定信息

没有官方依据时，不把“线路更贵”“IP 质量更高”等经验判断写成确定事实。网络表现应根据用户地区、运营商和时间实测。

## 联盟链接

部分链接包含 `aff=83193`。用户通过链接购买时，作者可能获得佣金。联盟关系不会改变官方来源优先级；文章会区分事实说明与购买入口。
